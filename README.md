# map-marker-labels

Create an app that allows users to create locations on top a full screen map by clicking on a location of the map + giving the location a name. Once created the location name should be displayed on top of the map as a draggable label.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/map-marker-labels.git
cd map-marker-labels
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
